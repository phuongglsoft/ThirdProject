## How To Use
FRONTEND:
```bash
# Clone this repository
$ git clone https://github.com/phuongglsoft/ThirdProject.git

# Go into the front end repository
$ cd ThirdProject/ThirdProject-FrontEnd

# Install dependencies
$ npm install

# Run the app
$ npm run dev
```
BACKEND:
```bash

# Go into the repository
$ cd ThirdProject

# Install dependencies
$ docker-compose up -d

```
